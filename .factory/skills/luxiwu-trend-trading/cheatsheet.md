# Cheatsheet — 趋势交易法快速决策指南

## 信号优先级（死规则）
| 优先级 | 信号 | 动作 |
|--------|------|------|
| 1 | 趋势线+A点同破 | 直接转向，任何工具不能否定 |
| 2 | 拐点线突破 | 80%反转概率 |
| 3 | RSI背离 | 减仓预警 |
| 4 | K线反转形态 | 入场确认，需其他工具配合 |

## 趋势方向判定
| 4H趋势 | 1H趋势 | 操作 |
|---------|---------|------|
| ↑ 上升 | ↑ 上升 | 做多 |
| ↑ 上升 | ↓ 下降 | 等1H调整完做多 |
| ↓ 下降 | ↓ 下降 | 做空 |
| ↓ 下降 | ↑ 上升 | 等1H反弹完做空 |

## 入场条件检查清单
- [ ] 4H定方向（不逆势）
- [ ] 波浪结构完整（5浪或7浪）
- [ ] 调整浪结构完整（abc/三角形/abcde）
- [ ] K线反转形态确认（鬼门关/启明星/黄昏星优先）
- [ ] 量化信号≥5分
- [ ] 止损在分界点A之外

## 仓位速算
仓位（手） = 每笔最大亏损（$） ÷ 止损点数
每笔最大亏损 = 总资金 × 30% ÷ 10

## 出场条件
| 条件 | 操作 |
|------|------|
| 到达PSTD目标 | 平仓1/2 |
| 趋势线+拐点线同破 | 平仓全部 |
| 反向浪1确立 | 平仓全部+转向 |
| RSI背离 | 减仓1/3 |

## 常见错误速查
| 错误 | 正确做法 |
|------|----------|
| RSI超买就做空 | 趋势中RSI超买=趋势强，不是反转信号 |
| 看到K线形态就入场 | 必须等结构完整+关键位置才有意义 |
| 止损放整数位 | 整数位容易被猎杀，放在整数外侧 |
| 亏损后加仓 | 永不逆势加仓 |
| 移动止损调远 | 止损只能收紧，不能调远 |

## 战胜股神：交易三步骤
1. **定方向** → 趋势线+拐点线+分界点A 判定上升/下降
2. **找入场** → 拐点或A点区域 + K线反转确认
3. **设出场** → 拐点线目标 + 资金管理止损

## 波浪理论使用原则
- 5浪/7浪是高概率框架，不是物理定律
- 其他结构形态也存在且有效
- 不因高频出现就排除其他可能性
- 保持批判：所有理论都是工具

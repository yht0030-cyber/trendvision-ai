# Chapter 8: 拐点线

## Core Idea
拐点线基于通道理论，用两条平行线包含趋势全部轨迹。价格与拐点线的交点=拐点，反转概率约80%（系统最高）。绘制口诀："要找下，先画上；要找上，先画下。"

## Frameworks Introduced
- **绘制方法**：
  - 上升趋势拐点线：先画上方外延线（连2+最外延高点）→平行下移至浪2低点
  - 下降趋势拐点线：先画下方外延线（连2+最外延低点）→平行上移至浪2高点

- **种类**：
  - 趋势拐点线：基于最大/耦合区间，突破=趋势结束
  - 导航拐点线：基于子浪区间，突破=子浪/调整浪完成

- **趋势线替代拐点线（五种场景）**：
  1. 假突破后创新高/低→改用趋势线
  2. 扩张趋势（δ=1.208-2.618）→趋势线替代
  3. 收缩趋势→以结构完整为信号，趋势线优于拐点线
  4. 钻石型→全程趋势线
  5. X型→全程趋势线

## Key Takeaways
1. 拐点线反转概率80%，系统最高
2. G4/G6拐点提供补仓信号
3. 短周期不用拐点线（转向过晚，盈利空间不足）
4. 过滤消息形成的假高/低点

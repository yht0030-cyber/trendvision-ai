# Chapter 7: 趋势线

## Core Idea
趋势线是最直观的趋势跟踪工具。连接同级别区间的高/低点，突破趋势线意味着70%概率趋势反转。趋势线+分界点A同时突破=系统最强信号。

## Frameworks Introduced
- **绘制规则（死规则）**：
  - 上升趋势线：连接最低点（或相对低点）与最高点之前的调整浪低点，中间不穿越任何价位
  - 下降趋势线：连接最高点（或相对高点）与最低点之前的调整浪高点，中间不穿越任何价位
  - 不能连接最高/最低点之后的点（除非价格回到78.6%位置）
  - 消息面造成的假高/低点可以过滤

- **趋势线种类**：
  - 主趋势趋势线：当前趋势最大Qjt与前反向最后Qjt同级别→突破=趋势反转
  - 导航趋势线：两个Qjt不同级别→突破只是浪1完成，不代表反转

- **直接转向 vs 延时转向**：
  - 直接转向：标准K线突破+距极值近（止损小）→立即平仓并反向建仓
  - 延时转向（三种）：非标准K线突破/突破点距极值过远/突破落入62%-78.6%位置

## Key Concepts
- **拐点线替代趋势线**：浪2回调≥78.6%或浪3加速过快→趋势线失去参照意义，改用拐点线
- **延时转向线**：取突破K线+后2根K线的低/高点画水平线，收市突破即执行

## Key Takeaways
1. 趋势线准确率约70%，不完美但可靠
2. 趋势线+分界点A同时突破→直接转向最强信号，任何工具都不能否定
3. 不能连接最高/最低点之后的点
4. 非标准K线突破必须延时确认

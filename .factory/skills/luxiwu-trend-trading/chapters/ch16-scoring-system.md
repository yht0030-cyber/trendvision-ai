# Chapter 16: 量化信号评分体系

## Core Idea
把三大工具+择时条件量化成 10 分制评分，回测与实盘共用同一套评分逻辑，确保信号客观可复现。这是项目脚本 trend_analyzer.py 把鹿希武理论自动化的核心，对应规则十九。

## Frameworks Introduced

- **10 分制结构（择时5 + 工具5）**：

  | 部分 | 项目 | 分值 |
  |------|------|------|
  | 基础择时(5) | 价格在支撑/阻力区 | 2 |
  | | RSI 位置(超卖恢复/超买回落) | 1 |
  | | K 线反转形态(最近3根内) | 2 |
  | 工具方向确认(5) | 拐点线突破 | 3 |
  | | 分界点A 突破 | 1 |
  | | 趋势线突破 | 1 |

- **拐点线权重最高(3分)**：回测准确率 81-93%，系统最高；趋势线最弱(1分)。

- **最强信号加成**：趋势线+分界点A 同向突破 → 额外 +1 分，且**无视基础分门槛直接触发**（对应规则十四）。

- **触发门槛**：做多/做空分数 ≥ 5 分。

- **4H 方向过滤（死规则）**：4H 下降→砍掉所有做多信号；4H 上升→砍掉所有做空信号。XAUUSD 加过滤后胜率 71%→90%。

## Key Concepts
- **score_from_window()**：回测+实盘统一入口，保证两边评分逻辑完全一致
- **价格位置**：基于价值区(value_area) ± ATR 判定 at_support / at_resistance
- **RSI 区间**：超卖恢复 30-55(做多+1)，超买回落 45-70(做空+1)
- **K线形态时效**：仅最近 3 根 K 线内的形态才计分

## K线关键位置过滤（2026-06-19 加入）

K 线反转形态的 2 分**仅在关键位置才计分**（第3版第四章：非关键位置的 K 线=噪音）：
- 关键位置 = 价值区边缘(支撑/阻力) **或** 价格贴近趋势线/拐点线/分界点A（1.5×ATR 内）
- 回测验证：总体胜率 68.9% → 71.6%，BTC 做多 43% → 75%，其余方向不降
- 脚本实现：evaluate_signal() 中 bull_pattern_valid / bear_pattern_valid

## Key Takeaways
1. 评分 ≥5 分才触发，趋势线+A点同破可无视门槛
2. 拐点线 3 分权重最高，趋势线 1 分最低
3. 4H 方向过滤是死规则，只做顺势
4. 回测与实盘共用 score_from_window()，不存在两套评分

# Chapter 14: RSI技术指标

## Core Idea
RSI(9)是趋势交易法的辅助工具，不能替代核心三工具（趋势线/拐点线/分界点A）。RSI的作用是"加分"而非"决策"。

## Frameworks Introduced
- **RSI辅助买入**：价格在调整浪末端+K线反转+RSI曾进入>70后回调→加分
- **RSI辅助卖出**：价格在调整浪末端+K线反转+RSI曾进入<30后反弹→加分
- **RSI背离**：价格创新高/低但RSI未配合进入超买/超卖→大级别调整预警，减仓不追单

- **三大禁忌**：
  1. 趋势中不能因RSI超买就反手做空/超卖就反手做多
  2. 没有三工具配合，RSI信号无意义
  3. 不因RSI未到位就拒绝系统信号

## Key Concepts
- **RSI(9)参数**：趋势交易法使用9周期RSI，比传统14周期更敏感
- **超买>70**：上升趋势中回调买入的参考，不意味做空
- **超卖<30**：下降趋势中反弹卖出的参考，不意味做多

## Key Takeaways
1. RSI永远是辅助，系统信号优先
2. RSI背离=减仓信号，不是反转信号
3. 趋势中RSI超买/超卖不代表可以反手

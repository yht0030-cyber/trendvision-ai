# Chapter 2: 波浪理论

## Core Idea
价格运动呈现波浪结构——主浪（推动浪）与调整浪交替推进。识别波浪结构是判断趋势阶段和预测反转时机的核心能力。

## Frameworks Introduced
- **趋势结构5种**：
  | 结构 | Qjt组合 | 反转概率 |
  |------|---------|---------|
  | 同级别5浪 | 2区间同级别 | 78% |
  | 扩张7浪 | 扩张+同级别/扩张 | 89% |
  | 收缩7浪 | 收缩+同级别/收缩 | 89% |
  | 钻石7浪 | 扩张+收缩 | 89% |
  | X7浪 | 收缩+扩张 | 89% |

- **"逢7必变"**：到达7浪结构，反转概率≥89%。这是系统最强结构信号之一。

- **奇偶浪法则**：奇浪（1/3/5/7）= 主浪；偶浪（2/4/6）= 调整浪。

## Key Concepts
- **主浪**：推动价格朝趋势方向运行的结构，分为简单主浪、复杂主浪、隐形主浪
- **调整浪**：在主浪之间运行的反向结构，种类包括简单、abc、三角形、abcde、钻石、X形
- **趋势级别**：长期(Qjt>45)、中期(Qjt 15-45)、短期(Qjt<15)
- **快慢节奏**：快升慢调(90%常见) vs 慢升快调(罕见，通常是abcde调整浪)

## Anti-patterns
- **忽视级别**：不区分趋势级别就数浪，导致多级别混淆
- **强行数到7**：结构未到7就强行预测反转

## Key Takeaways
1. "逢7必变"是概率≥89%的高确定性信号
2. 数浪必须先确定区间级别，同级别区间才能构成完整结构
3. 调整浪结构完整 + K线反转 = 入场时机
4. 快升慢调是常态，慢升快调警告大级别调整

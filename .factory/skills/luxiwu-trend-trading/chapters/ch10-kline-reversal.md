# Chapter 10: K线反转形态

## Core Idea
K线反转形态是入场时机的"扳机"。仅出现在关键位置（调整浪末端、趋势线/拐点线附近）的K线反转才有交易价值。

## Frameworks Introduced
- **标准K线定义**：取最近40根K线中最大5根计算均值。阳线收盘-最低>均值或阴线最高-收盘>均值→标准K线。标准K线用于减仓线调整和关键位置确认。

- **反转K线形态（仅关键位置有效）**：
  | 形态 | 信号 |
  |------|------|
  | 十字星 | 上下影2:1区分方向 |
  | 锤子线/倒锤子线 | 影线>1.618×实体 |
  | 阳抱阴/阴抱阳 | 实体完全覆盖前一根 |
  | 补缺影线 | 后实体覆盖前影线 |
  | 魂断蓝桥 | 光头对光脚 |
  | 启明星/黄昏星 | 趋势浪末端最强 |
  | 鬼门关★★★ | 标准K+十字星+反向标准K |
  | 急停加速★★ | 标准K+小K+同向标准K |
  | 海底捞月 | 假突破后快速收回 |

- **K线反转启动位置**：
  1. 主浪结构完整（7浪/同级别5浪）
  2. 调整浪结构完整（abc/abcde/三角形）
  3. 强40%回调位置
  4. 62%/78.6%回调位置

## Key Takeaways
1. 相同K线形态在非关键位置=噪音，在关键位置=信号
2. 鬼门关是系统最强K线反转形态
3. K线形态误报防护：吞没必须严格大于/小于，形态必须在最近2根K线内

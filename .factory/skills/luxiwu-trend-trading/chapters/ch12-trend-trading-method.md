# Chapter 12: 趋势交易法

## Core Idea
趋势交易法是一套基于概率与数理统计的完整交易系统。核心设计思想：不预测市场，只跟随趋势。成功率约48%，但通过3:1盈亏比实现长期正期望。

## Frameworks Introduced
- **顺势波段交易模型**（推荐新手）：
  1. 确认主趋势方向，浪1已形成
  2. 等待浪2调整完成（结构完整+K线反转）
  3. 浪2末端入场，被动止损设于浪1起始极值点之外
  4. 目标位用PSTD计算（1.208/1.618/2.000×H1）
  5. 到达目标且结构完整→平仓获利，等待浪4补仓
  6. 浪4末端补仓（半仓），目标位同PSTD

- **顺势连续交易模型**（有经验者）：
  - 复合头寸：每次入场2手，1手波段止盈，1手持至趋势反转
  - 主动止损随行情推进逐步收紧至导航A点
  - 突破趋势线/拐点线+结构完整→减仓1/2
  - 反向浪1确立→平仓全部并转向

- **假突破止损后处理**：
  - 被动止损被假突破→价格快速回到预期轨迹→立即补回头寸
  - 假突破后出现标准反向K线→确认假突破，可加码

## Key Concepts
- **系统平均准确率**：趋势线70%、拐点线80%、三工具综合48%
- **系统平均盈亏比**：2.9:1（接近3:1）
- **10次交易只需3次盈利即可盈亏平衡**（3:1盈亏比）

## Key Takeaways
1. 接受"近一半交易会亏损"是基本素养
2. 复合头寸管理：1手波段止盈+1手持至反转
3. 假突破止损后立即补回，不要犹豫

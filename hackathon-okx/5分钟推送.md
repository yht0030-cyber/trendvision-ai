# 🚀 只需 5 分钟！推送到 GitHub

## 方法一：解压zip推送（最简单）

1. 打开终端（Win+R → 输入 `cmd` → 回车）

2. 粘贴执行：
```bash
cd F:\投资\合约交易
```

3. 粘贴执行：
```bash
tar -xf hackathon-okx\trendvision-repo.zip -C .
```

4. 粘贴执行：
```bash
git push -u origin main
```

---

## 方法二：从头开始推（如果方法一不行）

打开终端，一条一条复制：

```bash
cd F:\投资\合约交易
git init
git config user.email "yht0030@gmail.com"
git config user.name "yht0030-cyber"
git add .
git commit -m "TrendVision AI v1.0 — OKX AI Genesis Hackathon"
git branch -M main
git remote add origin https://github.com/yht0030-cyber/trendvision-ai.git
git push -u origin main
```

> 推送时如果有登录窗口，点「登录」，浏览器已经登录过 GitHub 了，不会弹密码

---

## 方法三：直接把文件拖进去

如果都不会，最简单的：
1. 打开 https://github.com/yht0030-cyber/trendvision-ai
2. 点 **"Add file"** → **"Upload files"**
3. 把 `F:\投资\合约交易\hackathon-okx\` 里的文件 + `scripts\` 文件夹拖进去
4. 点 **"Commit changes"**

---

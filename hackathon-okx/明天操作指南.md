## 🎯 第1步：推送到 GitHub（2分钟）

**打开终端**（`F:\投资\合约交易` 目录下），一条一条复制执行：

```bash
git init
git config user.email "yht0030@github.com"
git config user.name "yht0030-cyber"
git add .
git commit -m "TrendVision AI v1.0 — 基于趋势交易理论的AI量化分析系统"
git branch -M main
git remote add origin https://github.com/yht0030-cyber/trendvision-ai.git
git push -u origin main
```

> 如果弹出 GitHub 登录窗口，点「登录」→ 浏览器里登录过的，直接授权就行
> 如果提示输入密码，去 https://github.com/settings/tokens 生成 token（选 repo 权限），密码位置粘贴 token

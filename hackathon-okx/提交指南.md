## 你需要做的事（明天起床后）

### Step 1: 注册比赛

```
https://www.hackquest.io/zh-cn/hackathons/OKXAI-Genesis-Hackathon
```

### Step 2: 提交 GitHub 仓库
报名成功后，找到 **"提交项目" / "Submit Project"** 按钮，填入：
- **项目名称**: TrendVision AI
- **项目简介**（30字以内）: 基于趋势交易理论的AI量化合约分析系统，一键输出方向/评分/入场/止损
- **GitHub 链接**: https://github.com/yht0030-cyber/trendvision-ai
- **项目描述**（长文）:

```
TrendVision AI 是一套将经典技术分析理论（趋势交易理论）完整代码化的AI量化分析系统。

核心能力：
- 双周期验证：1H + 4H 双周期确认，4H方向过滤死规则
- 量化评分10分制：≥5分触发信号，趋势线+分界点A同时突破为最强信号
- 一键监控：python monitor.py BTC 一条命令3秒出结论
- 回测验证：XAUUSD 90%胜率、BTC 81%胜率、ETH 70%胜率

技术栈：Python + CCXT + 经典技术分析理论体系
```

### Step 3: 提交演示视频（可选但加分）
